07/10/2007
Pentaho metadata Release Notes for: 1.5.5

This is a milestone\development build of the Pentaho metadata layer and the Metadata Editor. 
All major features are complete.  Many usability improvements and bugs have been fixed since 
Milestone 4.  This milestone build is not production ready.

Release History
===============
